# ChatKit React

This package provides React bindings for ChatKit, offering lightweight components and hooks to integrate ChatKit into React apps.

## Role in the ChatKit project

- Wraps core ChatKit APIs with idiomatic React components and hooks for faster integration.
- Keeps UI-facing types aligned with `@xpertai/chatkit` and `@xpertai/chatkit-types` without bundling implementation details.
- Serves as the React-specific entry point so applications can adopt ChatKit with minimal setup.
